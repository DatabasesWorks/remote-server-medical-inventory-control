Info About These Docs

General

These documents are generated using zola with a fork of the Adidoks theme. The documention content is located in the ./docs/content/docs folder of the Open mSupply Remote Server repository.

To run a local version of these docs while developing:

  • Install zola
  • Pull the theme sub module by navigating to the theme folder and running (git submodule init && git submodule update)
  • Run with zola serve, and open localhost:1111 in your browser.

Alternatively you can use a dockerised version

Syntax Highlight

Zola's default markdown syntax highlight parser grouped a lot of graphQL tokens, and conflicted with our theme styles so that code blocks were hard to see, so we've changed to using prismjs. This required a little bit of overriding. We did the following:

  • Extended the base page.html to include prism.js prism.css and theme_override.css.
  • Generated prism.js and prism.css with default styles for the languages listed below, using Prism's custom download tool, and added those files to the static folder
  • Added an override of some styles, as described here

Adding new Languages

In the bundles js and css there are Prism syntax highlight languages for:

  • javascript
  • graphql
  • json
  • rust
  • typescript
